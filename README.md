# FRMC 

Last update: v1.0.0

This is a fast and robust imputation software based on matrix completion, 
called FRMC, for single cell RNA-Seq data .


## Requirements
* Python 3
* numpy
* scipy
* matplotlib
* sklearn
* pandas
* math


## Usage
```
python FRMC.py <Normalized Matrix file(cell-by-gene)> <out_prefix>

```
